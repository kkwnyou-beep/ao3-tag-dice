# 🎲 AO3 Tag Dice

一个轻量级的 AO3 tag 随机生成器，帮助创作者快速获得创意灵感。从多个维度随机抽取 tag，组合出独特的创意题材。

## ✨ 功能特性

- 🎲 **完整 Roll**：从每个维度随机抽取一个 tag
- 📌 **预设模式**：快速加载预定义的 tag 组合（甜饼、虐心、高甜等）
- 📋 **一键复制**：生成 `#tag1 #tag2 ...` 格式的文本
- 🔄 **重新 Roll**：快速刷新结果
- 🗑 **清空**：重置所有选择
- 📦 **完全可扩展**：所有数据在 `tags.json` 中，无需修改代码就能扩展

## 📊 数据结构

所有 tag 数据存储在 `tags.json` 中，包含两部分：

### 1. 维度（dimensions）

```json
{
  "dimensions": {
    "场景": ["咖啡馆", "图书馆", ...],
    "情绪": ["温柔", "克制", ...],
    "关系张力": ["青梅竹马", "宿敌", ...],
    "动作": ["壁咚", "亲吻额头", ...],
    "限制条件": ["AU", "现代", ...],
    "强度": ["轻度", "中度", ...],
    "额外氛围": ["下雨", "烛光", ...]
  }
}
```

每个维度是一个数组，Roll 时会从中随机选取一个。

### 2. 预设（presets）

```json
{
  "presets": {
    "甜饼": {
      "场景": "咖啡馆",
      "情绪": "温柔",
      "关系张力": "青梅竹马",
      ...
    },
    "虐心": { ... }
  }
}
```

点击预设按钮可快速加载预定义的 tag 组合。

## 🛠 如何扩展

### 添加新的 tag

编辑 `tags.json`，在对应维度的数组中添加新 tag：

```json
"场景": [
  "咖啡馆",
  "图书馆",
  "你的新 tag"  // ← 添加到这里
]
```

### 添加新的维度

在 `tags.json` 的 `dimensions` 中添加新维度：

```json
"dimensions": {
  "场景": [...],
  "情绪": [...],
  "你的新维度": ["tag1", "tag2", "tag3"]  // ← 新维度
}
```

### 添加新的预设

在 `tags.json` 的 `presets` 中添加：

```json
"presets": {
  "甜饼": { ... },
  "你的预设名": {
    "场景": "具体值",
    "情绪": "具体值",
    ...
  }
}
```

添加后，页面会自动生成对应的预设按钮！

## 📱 特性支持

- ✅ 响应式设计，支持移动设备

## 🐛 已知限制

- 浏览器必须支持 `fetch` API 和 `navigator.clipboard`
- 本地文件系统打开时可能因 CORS 限制无法加载 `tags.json`（建议用本地服务器）
